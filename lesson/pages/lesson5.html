GIT flow

GIT clone - для того, чтоби "склонить" репозиторий
GIT pull - для тогго, чтоби получить изменения с репозитория
GITcheckout -b branchname - создать новую ветку и переключиться на нее
GIT add - добавить изменения, которие сделани локально
GIT commit -m "commit text" - подписать их
GIT push origin branchname - запушить на git сервер
